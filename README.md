# Multi Timer Demo

This project is a simple demo webpage that displays **four independent timers** arranged in a responsive grid. Each timer can be started, stopped, and reset, and includes an adjustable alarm limit.

## Features

- **Four independent timers** labeled *Sheet 1â€“4*
- Timers count in **hours:minutes:seconds** (up to 3 hours)
- **Start / Stop / Reset** controls
  - Reset only works if the timer is stopped
- **Adjustable alarm limit** (hours + minutes)
- **Flashing numbers** when within 5 minutes of the alarm limit
- **Red display + buzzer sound** when the alarm limit is reached
- Responsive **four-quadrant layout** that scales to fit the display
- Clean, simple design with a sans-serif font

## Demo

Open `index.html` in any modern browser to run the demo locally.
